# Project Images

Place your project images in this directory with the following names:

- `ctmp.jpg` - Constant-Time Motion Planning (CTMP) for Robotic Manipulators
- `swarm-planning.jpg` - Swarm Planning with Adaptive Map Discretization
- `finde.jpg` - FinDe++: Augmented Decoder-Only Model for Financial Forecasting
- `rl-algorithms.jpg` - Reinforcement Learning Algorithms: From TD Learning to Hierarchical Methods
- `property-inspection.jpg` - Automated Property Inspection with Computer Vision
- `cryogenic-device.jpg` - Cryogenic Bone Tumor Removal Device

**Recommended image specifications:**
- Format: JPG or PNG
- Size: 800x600 pixels or similar aspect ratio (4:3)
- File size: Under 500KB for optimal loading

If an image is not found, the component will gracefully hide the image placeholder.
